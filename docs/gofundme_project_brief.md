# GoFundMe AI Accelerated Project Brief

**Project Title:** Build GoFundMe Profile, Fundraiser and Community Pages

**Project Category:** AI Accelerated

## Problem Statement

*   Create an AI-powered experience that reimagines how GoFundMe profiles, fundraiser pages, and community pages work together.
*   Use AI creatively to design engaging versions of the following page types:
    *   Profile Page
    *   Fundraiser Page
    *   Community Page
*   Pages should feel connected and provide a seamless user experience.

## Example References

*   Fundraiser Example: https://www.gofundme.com/f/realtime-alerts-for-wildfire-safety-r5jkk
*   Community Example: https://www.gofundme.com/communities/watch-duty
*   Profile Example: https://www.gofundme.com/u/janahan

## Business Context

*   Reimagine GoFundMe as a more engaging destination for donors and supporters.
*   Enable people to unite around causes they care about.
*   Encourage users to return frequently through meaningful engagement.

## Impact Metrics

*   Repeat Visits
*   Meaningful Actions (Donate, Share, Follow)

## Technical Requirements

*   Languages: Developer's choice
*   Framework: React / Next.js preferred
*   Cloud Platform: AWS preferred

*   Development approach: Use AI where possible to accelerate the build

## Success Criteria
*   Pages load successfully with fast response times
*   Profile, Fundraiser, and Community pages are intuitive and seamlessly integrated
*   Instrumentation and analytics included (what metrics you track and why)

Page load times should be optimized.
